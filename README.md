# ☕︎ Calculadora de Cargas Elétricas

Calculadora de Cargas Elétricas é uma aplicação web simples e estilosa que permite calcular a razão entre cargas elétricas em um sistema, com interface moderna e responsiva. Tudo feito com HTML, CSS e JavaScript. Ideal para estudantes ou curiosos por física e programação.

## ☕︎ Funcionalidades

- Entrada de valores para a, x₃ e q₂;
- Validação dos dados de entrada;
- Cálculo e exibição da razão com fórmulas matemáticas renderizadas via MathJax.

# ☕︎ Modo de Operação

- Modo Cálculo: Insira parâmetros conhecidos para obter resultados.

## ☕︎ Efeitos Visuais

- Interface com efeito Glassmorphism;
- Troca dinâmica entre o tema claro e escuro com ícones de sol e lua;
- Botão "Calcular" com brilho animado que muda conforme o tema ativo;
- Fundo em degradê suave que transita entre as paletas claras e noturnas;
- Visualização matemática das fórmiças com MathJax para melhor legibilidade.

## ☕︎ Tecnologias Utilizadas 

- HTML5;
- CSS3 (incluindo efeitos visuais e animação);
- JavaScript (validação, cálculo e controle do tema);
- MathJax para renderizar fórmulas matemáticas em LaTex.

## ☕︎ Fórmulas Utilizadas

- **Razão entre as cargas (q₁/q₂):**  
  `q₁ / q₂ = ((x₃ + a) / (x₃ - a))²`

> Onde:  
> - `a` é o parâmetro fornecido pelo usuário;  
> - `x₃` é a posição relativa da carga;  
> - `q₂` é a carga de referência.
